Gen HTML Shell Script

Hello, I will telling you about a script I wrote to auto generate an HTML boilerplate filled file.


Code

            #!/bin/sh
            TARGET=$1
            SOURCE=template.html
            cat $SOURCE > $TARGET

The above code is the content of the gen-html.sh script in this project which I use to generate all my HTML files before I start working

Variables:

  • TARGET: This variable refs the path to which I wish to write the boilerplate to.
  • SOURCE: This variable refs the source of the boilerplate
